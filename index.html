mini project

카페 키오스크 개발

기술 스택 : java,java swing,oracle
주요 기능 : 카페 키오스크는 카페의 메뉴를 버튼 또는 리스트로 표시하고, 사용자가 메뉴 항목과 수량을 선택하여 주문을 작성할 수 있도록 합니다. 주문이 완료된 후에는 결제 방법을 선택하고 결제를 처리하며, 이전 주문 내역을 확인할 수 있는 기능도 제공합니다.
내 역할 : java Swing을 사용하여 사용자가 메뉴를 쉽게 선택할 수 있도록 직관적인 UI를 설계하고 구현했습니다.
성과: 사용자가 메뉴를 쉽게 선택하고 결제까지 빠르게 진행할 수 있어 직관적인 UI 구현으로 사용자 경험 개선을 시켰습니다.

Devhub

 Github  Notion
초보 개발자를 위한 형상관리 및 코드비교 서비스
기술 스택 : java, HTML, CSS, Javascript, Spring Boot, MySQL, REST API, BootStrap
주요 기능 :
  • 회원 관리: 로그인, 로그아웃, 회원가입, 이메일 인증, 소셜 로그인 (구글)
  • 나의 프로젝트: 저장소 이름 및 설명 관리, 저장소 수정 및 삭제
  • 팀 프로젝트: 프로젝트 생성, 목록 조회, 참가, 팀원 초대, 레포지토리 생성, AI 코드 리뷰, 코드 검색
  • 쪽지: 쪽지 보내기, 받은 쪽지함, 보낸 쪽지함
  • AI 도우미: 자동화된 작업 처리, 대화 내역 저장
  • 팀원 구하기: 게시판 생성, 수정, 삭제, 게시판 목록 및 내 게시판 조회
  • 마이 페이지: 비밀번호 변경, 프로필 사진 변경, 진행 사항 조절, 기록 조회, 팀 목록, 게시글 기록, 쪽지함, 결제 내역
내 역할 :
  • 게시판 생성
  • 게시판 수정
  • 게시판 삭제
  • 게시판 목록 조회
  • 내 게시판 목록 조회
성과 : 게시판의 생성(Create), 조회(Read), 수정(Update), 삭제(Delete) 기능을 REST API로 구현하여 간결하고 확장 가능한 구조를 만들었습니다.
